# claude-plugins

Franco Gilio's plugin marketplace for [Claude Code](https://code.claude.com).
Add it once and install any of the plugins below.

## Usage

```
/plugin marketplace add fgilio/claude-plugins
/plugin install coding@fgilio
```

## Plugins

- **coding** ([fgilio/coding-skill](https://github.com/fgilio/coding-skill)):
  code style preferences. Expressive naming, fluent interfaces, early returns,
  Laravel-style comments, Stripe-style docs.
- **skill-design-philosophy** ([fgilio/skill-design-philosophy-skill](https://github.com/fgilio/skill-design-philosophy-skill)):
  design philosophy for creating outstanding Claude Code skills.

## Updates

Plugins track the latest commit of their source repositories. Claude Code
checks for updates on startup. Run `/plugin update <name>` to update manually,
or enable auto-update for this marketplace in the `/plugin` menu.
